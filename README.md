# 微信小程序－技能图谱

### 说明：

实现技能列表，技能详情等功能，特色功能：
- 使用markdown解析数据

### 数据接口:

- https://api.getweapp.com/thirdparty/github/skill-map

### 目录结构：

- resources — 存放项目图片文件
- pages — 存放项目页面相关文件，包括about,index,logs,skill,skill-list,test等页面
- store — 存放数据接口文件
- utils — 存放markdown,时间格式化等文件

### 开发环境：

微信web开发者工具 v0.10.102800
